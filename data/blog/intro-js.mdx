# ¿Qué es JavaScript ?

JavaScript es uno de los lenguajes de programación más ampliamente utilizados y populares en el mundo. 

Su inicio se dio con el propósito de agregar interactividad y funcionalidad a las páginas web. Sin embargo, ha crecido significativamente y se ha transformado en un lenguaje de programación de <strong>propósito general</strong>. 

Su flexibilidad y la extensa comunidad de desarrolladores que lo respalda hacen que JavaScript continúe siendo un <strong>pilar fundamental en la industria tecnológica</strong>.

> Programar es el acto de crear un conjunto de instrucciones, también conocido como "código", para guiar a una computadora u otro 
> dispositivo en cómo llevar a cabo una tarea específica. 

## Por qué aprender JavaScript


JavaScript es el <strong>único</strong> lenguaje de programación que todos los navegadores web comprenden de forma nativa. Por eso, prácticamente todas las páginas web que visitamos <strong>contienen alguna línea de código de JavaScript</strong>. 


<strong>Su curva de aprendizaje</strong> es bastante corta ya que en muy
poco tiempo puedes empezar a hacer desarrollos y aplicaciones complejas, aunque para llegar ser un buen
programador es necesario una gran cantidad de años de experiencia con el mismo.

JavaScript es altamente solicitado como lenguaje de programación. Se emplea ampliamente en aplicaciones web, móviles, escritorio, backend, videojuegos, inteligencia artificial, y más. <strong>Ofrece un amplio abanico de oportunidades en un solo lenguaje</strong>.


## La consola del navegador

Todos los navegadores presentan una herramienta llamada consola. La consola nos <strong>permite ejecutar código JavScript en tiempo real y ver el resultado</strong>. También ahí podemos ver los erroes, advertencias y trazas que se producen en nuestro código.

### Cómo abrir la consola

Para abrir la consola del navegador debes hacer lo siguiente: 

* Abre tu navegador web favorito, por ejemplo, Google Chrome. 
* Ve a la página about:blank en la barra de direcciones. Así evitaremos que la página web que hemos cargado nos moleste.
* Haz click derecho en cualquier parte de la página y selecciona la opción Inspeccionar Elemento o Inspect.

Aquí podrás ejecutar tu código JavaScript y ver el resultado. Es lo que usaremos en las primeras clases del curso. Más adelante pasaremos a usar un editor de código más potente.
