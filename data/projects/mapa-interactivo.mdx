## Definición

La aplicación o proyecto consiste en la implementación de un mapa intecactivo haciendo de la api <HighlightedCode>MapBox</HighlightedCode> y <HighlightedCode>React</HighlightedCode>.
Este conjunto de tecnologías permite una experiencia de usuario fluida y atractiva.

Es un <HighlightedCode>proyecto básico</HighlightedCode> que sirve como punto de partida. Aunque es simple en la actualidad, su simplicidad permite futuras mejoras y funcionalidades más avanzadas. Las tecnologías utilizadas, como React, Bootstrap y Mapbox, brindan una base sólida para futuros desarrollos que podrían convertir este proyecto en algo más completo y versátil.

El proyecto se puede dividir:

- <HighlightedCode>Estructura: Componentes React</HighlightedCode>
- <HighlightedCode> Lógica/Funcionalidad</HighlightedCode>
- <HighlightedCode> API</HighlightedCode>

## Estructura: Componentes React

La estructura del proyecto se ha desarrollado utilizando <HighlightedCode>React</HighlightedCode> y se han usado componentes para dividir el proyecto en diferentes partes. Esta técnica o patrón permite una mayor organización y escabilidad del proyecto.

El <HighlightedCode>Componente principal</HighlightedCode> se llama <HighlightedCode>HomeScreen.tsx</HighlightedCode>. Este componente es el <strong>end point</strong> de entrada a la aplicación y es donde se han agregado el resto de los componentes que conforman la aplicación.

<CodeBlock text={`
import { 
    BtnMyLocation, 
    MapView, SearchBar, 
    ViteLogo 
} from "../components";

export const HomeScreen = () => {
  return (
    <>
      <MapView />
      <BtnMyLocation />
      <ViteLogo />
      <SearchBar />
    </>
  );
};
`}  />

Los componentes incluidos son los siguientes:

- <strong>MapView.tsx:</strong> Componente que se encarga de renderizar la vista
  del mapa.
- <strong>BtnMyLocation.tsx:</strong> Botón de localización por defecto.
- <strong>ViteLogo.tsx: </strong> Logo de vite.
- <strong>SearchBar.tsx:</strong>Componente para buscar una determinada
  localización.

## Lógica/Funcionalidad

La lógica de la aplicación se ha desarrollado utilizando React y ContextAPI

> <HighlightedCode>ContextAPi</HighlightedCode> es una <HighlightedCode>
>   API
> </HighlightedCode> de React que permite el manejo del estado de nuestra aplicación.
> Esto permite que los datos se puedan compartir entre componentes sin tener que
> pasarlos manualmente a través de los <HighlightedCode>props</HighlightedCode>.

Concretamente, se han creado dos contextos para manejar el estado de la aplicación:

- <HighlightedCode>MapContext.ts:</HighlightedCode> En este contexto se controla
  el estado del mapa: los cambios en mapa, si está cargado y polylines entres dos
  puntos del mapa.
- <HighlightedCode>PlacesContext.ts:</HighlightedCode> En este contexto se controla
  el estado del punto seleccionado en el mapa: carga, localización del usuario y
  todos los posibles lugares.

Además de estos contextos, que se encargan de controlar el estado de la aplicación, la aplicación presenta las siguientes funcionalidades:

- <strong>Botón de ubicación por defecto</strong>
  <br /> Esta funcionalidad nos permite volver a la localización por defecto del
  usuario, es decir, al clicker el botón el mapa se reajusta y va hacía la posisicón
  en la que se encuentra el dispositivo que está usando el usuario para interactuar
  con el mapa. En caso, de qué el usuario no tenga un dispositivo con geolocalización
  no podrá hacer uso de esa aplicación.
- <strong>Cajetín de búsqueda</strong>
  <br /> Esta funcionalidad nos permite buscar una determinada localización y que
  en el mapa de pinten (marcadores) los diferentes resultados que se han obteneido.
  Todos estos resultado estarán relacionados con la cadena o la información que se
  ha introducido en el cajetín de búsqueda.

- <strong>Botones direcciones </strong> <br />
  Esta funcionalidad nos permite pintar una línea entre el punto donde nosotros
  estamos localizados y la dirección o punto que hemos seleccionado.{" "}

## Api: MapBox

<HighlightedCode>MapBox</HighlightedCode> es una plataforma de mapeo y geolocalización
que proporciona una variedad de herramientas y servicios para trabajar con mapa e
integrar datos geoespaciales en aplicaciones y sitios web. Esta plataforma ofreece
una amplica gama de funcionalidades para desarrolladores.{" "}

En este proyecto se han hecho uso de las siguientes funcionalidades que nos proporciona la api:

- <strong>Visualización de Mapas Personalizados</strong>
- <strong>Interacción con los Maps</strong>
- <strong>Geolocalización y Búsqueda</strong>
- <strong>Enrutamiento y Direcciones</strong>
- <strong>Datos en tiempo real</strong>
