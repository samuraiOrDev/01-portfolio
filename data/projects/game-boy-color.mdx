# Creación de un GameBoyColor utilizando solo CSS

<ComponentDate date="04-07-2023" time="5 minutos" />
<Link href="https://gameboy-color.vercel.app/" />

## Definición

El proyecto <HighlightedCode>GameBoyColor</HighlightedCode> se encarga de demonstrar la versatibilidad y el poder de las hojas de estilos en cascada al recrear un icónico dispostivo de juego, exclusivamente mediante técnicas de diseño CSS.

En este proyecto, se ha logrado emular con precisión la apariencia y la estructura de la GameBoyColor, sin utilizar ninguna imagen o elemento gráfico (a excepción de pikachu), lo que destaca la capacidad de CSS para crear diseños complejos y detallados

### Desarrollo

El proyecto se ha realizado al completo usando <HighlightedCode>CSS</HighlightedCode> Vanila para los estilos de la GameBoyColor y <HighlightedCode>HTML</HighlightedCode> para la estructura de la misma.
A continuación, se describen los principales pasos que se siguieron en el proceso de desarrollo:

- <strong>Estructura HTML:</strong> El proyecto comenzó con la estructura básica
  de HTML que estableció la disposición general del GameBoyColor. Se crearon elementos
  div para representar la carcasa, la pantalla, los botones y otros componentes del
  dispositivo.

- <strong>Estilización de la carcasa:</strong> Mediante el uso de propiedades como
  <b>border</b>,<b>border-radius</b> y <b>box-shadow</b>, se diseñó la carcasa
  exterior del GameBoyColor. Se aplicaron colores y gradientes para lograr la
  apariencia auténtica de la consola.

- <strong> Pantalla y visualización:</strong> Para la pantalla principal del GameBoyColor,
  se utilizó CSS para crear un rectángulo que emula la pantalla LCD. Se ajustaron
  las dimensiones y se aplicaron sombras y reflejos para simular la iluminación de
  la pantalla.

- <strong>Botones y controles:</strong> Los botones direccionales y los botones
  <HighlightedCode>A</HighlightedCode> y <HighlightedCode>B</HighlightedCode> fueron
  creados utilizando combinaciones de elementos <b>div</b> y pseudoelementos
  <b>::before</b> y <b>::after</b>. Se aplicaron colores y sombras para dar
  profundidad y realismo a los botones.

- <strong>Detalles y logotipos:</strong> Se agregaron detalles finos como el logotipo
  de Game Boy Color y otras marcas distintivas utilizando propiedades de texto y
  posicionamiento preciso.

- <strong>Resolución responsive:</strong> Se implementó una respuesta al tamaño de
  la pantalla utilizando consultas de medios, lo que permitió que el diseño se adaptara
  de manera coherente en diferentes dispositivos y tamaños de pantalla.
